atto_styles
=================

This is a plugin for Atto editor in Moodle that allows toggles the display of
simple markup inside the HTML editor mode. To install unpack to lib/editor/atto/plugins/styles, 
and visit the admin notifications page to install.  Then add the button
on the Atto toolbar settings page (maybe next to html).
